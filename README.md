# DSA-Super50

DSA-Super50 is a C++ repository focused on building strong fundamentals in Data Structures and Algorithms. It contains beginner-friendly implementations along with practice problems commonly used in competitive programming.

## Technologies Used

- Language: C++
- Compiler: GCC / MinGW
- IDE: Visual Studio Code

## How to Run

Compile any file using:
g++ filename.cpp -o filename

Run the executable:
./filename

## Purpose

This repository is intended for:
- Learning DSA concepts
- Practicing C++ programming
- Preparing for coding interviews and contests
