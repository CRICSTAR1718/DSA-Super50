Exponential Decay
Difficulty: MEDIUM | Max Score: 50
A function f(x) is defined for non-negative integers x and satisfies the following conditions:

f(0) = 1
For any positive integer k,
 f(k) = f(⌊k / 2⌋) + f(⌊k / 3​⌋)

where ⌊x⌋ denotes the greatest integer less than or equal to x.



Given a non-negative integer N, your task is to compute the value of f(N).



Input Format

The input consists of a single integer N.


Output Format

Print a single integer — the value of f(N).


Constraints

N is an integer satisfying 0 ≤ N ≤ 1018


Example 1:

Input: 2
Output: 3
Explanation: We have 
f(2)=f(⌊2/2⌋)+f(⌊2/3⌋) = f(1) + f(0) = (f(⌊1/2⌋) + f(⌊1/3⌋)) + f(0) = (f(0) + f(0)) + f(0) = 3.
