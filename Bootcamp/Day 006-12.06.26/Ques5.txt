Path Sum (Medium Version)
Difficulty: MEDIUM | Max Score: 50
You are given the Root of Binary tree and an integer target.

Your task is to return all root-to-leaf paths where the sum of the node values in the path equals target. 

Each path should be returned as a list of the node values, not node references.

A root-to-leaf path is a path starting from the root and ending at any leaf node. A leaf is a node with no children.



 

Example:

Input: target = 14
         5
        / \
      -2   5
     /  \   \
    11   2    4
Output: [[5,-2,11],[5,5,4]]
Explanation: 5-2+11 = 14
             5+5+4 = 14
 

Constraints:

0 <= Number of Nodes in the tree <=104
-104 <= node values, target <= 104
